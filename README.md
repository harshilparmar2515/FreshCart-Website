# FreshCart-Website
FreshCart UI is a responsive grocery eCommerce homepage built using HTML5 and CSS3. The project replicates a modern online grocery store interface featuring product grids, promotional banners, daily deals, and a structured footer. It focuses on layout design, reusable components, hover interactions, and clean visual hierarchy.
